# Restaurant Menu Page

A simple, interactive restaurant menu web page built with HTML, CSS, and JavaScript.

## Features

- **Menu Categories**: Organized sections for Appetizers, Main Courses, Desserts, and Drinks
- **Interactive Cart**: Add items to a shopping cart with quantity tracking
- **Responsive Design**: Clean, mobile-friendly layout
- **Smooth Navigation**: Jump to different menu sections

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling for the menu page
- `re.js` - JavaScript for cart functionality

## Usage

1. Open `index.html` in a web browser
2. Browse the menu items
3. Click "Add to Cart" to add items to your order
4. View your cart in the top-right corner

## Customization

- Edit `index.html` to change menu items and prices
- Modify `style.css` to update colors and layout
- Enhance `re.js` to add more features like checkout or item removal

## Browser Support

Works in all modern web browsers that support ES6 JavaScript.
